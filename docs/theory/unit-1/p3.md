# Unit 01 — Part 3

# Swing Components: JApplet, JFrame, JLabel, JTextField, JTextArea, JButton, JCheckBox, JRadioButton, JComboBox, JMenu

---

## 1. JApplet

- Swing version of `Applet`, used to build GUI applications that run inside a web browser or applet viewer.
- Extends `java.applet.Applet`.
- Supports Swing's lightweight components (plain `Applet` supports only AWT).
- Has lifecycle methods: `init()`, `start()`, `stop()`, `destroy()`.

**Syntax:**
```java
class MyApplet extends JApplet {
    public void init() {
        // add components here
    }
}
```

**Example:**
```java
import javax.swing.*;

public class MyApplet extends JApplet {
    public void init() {
        JButton button = new JButton("Click Me");
        getContentPane().add(button);
    }
}
```

**Remember:**
- Applets are now obsolete in modern browsers (no Java Plugin support).

## 2. JFrame

- Top-level container representing a standalone window (title bar, borders, minimize/maximize/close buttons).
- Every standalone Swing application needs at least one `JFrame`.
- Extends `java.awt.Frame`. `JFrame` is the root component of the Swing hierarchy.
- For using it we import `javax.swing.JFrame`.

**Syntax:**
```java
JFrame frame = new JFrame(String title);
frame.setSize(int width, int height);
frame.setLayout(LayoutManager mgr);
frame.setVisible(boolean b);
```

**Example:**
```java
import javax.swing.JFrame; // or import javax.swing.*;
JFrame frame = new JFrame("My Window");
frame.setSize(400, 300);
frame.setLayout(new FlowLayout());
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setVisible(true);
```

::: warning **Remember:**
- `setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE)` is needed, otherwise the program keeps running even after the window is closed.
- `setVisible(true)` should be called **last**.
:::


## 3. JLabel

- Displays non-editable text, an image, or both.
- Does not generate any events (purely for display).

**Syntax:**
```java
JLabel label = new JLabel(String text);
```

**Example:**
```java
import javax.swing.JLabel; // or import javax.swing.*;

JLabel label = new JLabel("Username:");
```

---

## 4. JTextField

- Allows the user to enter/edit a **single line** of text.
- Generates an `ActionEvent` when Enter is pressed.

**Syntax:**
```java
JTextField textField = new JTextField(int columns);
```

**Example:**
```java
import javax.swing.JTextField; // or import javax.swing.*;
JTextField textField = new JTextField(15);
add(textField);
```

::: warning **Remember:**
- `JPasswordField` extends `JTextField` and masks the entered characters — used for passwords. 
```java
import javax.swing.JPasswordField;
JPasswordField passwordField = new JPasswordField(15);  // for passwords
```
:::

## 5. JTextArea

- Allows the user to enter/edit **multiple lines** of plain text.
- Does not generate `ActionEvent` on Enter — Enter just creates a new line.

**Syntax:**
```java
JTextArea textArea = new JTextArea(int rows, int columns);
```

**Example:**
```java
import javax.swing.JTextArea;
import javax.swing.JScrollPane;
// or import javax.swing.*;

JTextArea textArea = new JTextArea(5, 20);
JScrollPane scrollPane = new JScrollPane(textArea);
```

::: tip **Remember:**
- Usually wrapped inside a `JScrollPane` to allow scrolling when text exceeds visible area.
:::

## 6. JButton

- Creates a clickable push button.
- Generates an `ActionEvent` when clicked, handled using `ActionListener`.

**Syntax:**
```java
JButton button = new JButton(String text);
button.addActionListener(ActionListener l);
```

**Example:**
```java
import javax.swing.JButton;

JButton button = new JButton("Submit");
button.addActionListener(e -> System.out.println("Button Clicked!"));
```


## 7. JCheckBox

- Represents a box that can be checked (selected) or unchecked (deselected).
- Multiple checkboxes can be selected independently at the same time.
- Generates an `ItemEvent` when toggled.

**Syntax:**
```java
JCheckBox checkBox = new JCheckBox(String text, boolean selected);
```

**Example:**
```java
import javax.swing.JCheckBox;
JCheckBox cb1 = new JCheckBox("Java");
JCheckBox cb2 = new JCheckBox("Python", true);
```

## 8. JRadioButton

- Two-state button (selected/unselected), used in groups where only **one** option can be selected.
- Must be grouped using `ButtonGroup` to enforce single selection.

**Syntax:**
```java
JRadioButton radio = new JRadioButton(String text);
ButtonGroup group = new ButtonGroup();
group.add(radio);
```

**Example:**
```java
import javax.swing.JRadioButton;
import javax.swing.ButtonGroup;

JRadioButton male = new JRadioButton("Male");
JRadioButton female = new JRadioButton("Female");

ButtonGroup group = new ButtonGroup();
group.add(male);
group.add(female);
```
::: warning **Remember:**
- Without adding to a `ButtonGroup`, multiple radio buttons can be selected together (defeats their purpose).
:::

## 9. JComboBox

- Dropdown list allowing the user to select **one item** from a list of options.
- Saves screen space compared to showing all options at once.

**Syntax:**
```java
JComboBox<String> comboBox = new JComboBox<>(String[] items);
```

**Example:**
```java
import javax.swing.JComboBox;
String[] courses = {"Java", "Python", "C++"};
JComboBox<String> comboBox = new JComboBox<>(courses);
```

## 10. JMenu

- Creates a dropdown menu (like File, Edit, Help) that holds a list of `JMenuItem`s.
- Must be added to a `JMenuBar`, which is added to a `JFrame`.

**Syntax:**
```java
JMenuBar menuBar = new JMenuBar();
JMenu menu = new JMenu(String text);
JMenuItem item = new JMenuItem(String text);
menu.add(item);
menuBar.add(menu);
frame.setJMenuBar(menuBar);
```

**Example:**
```java

import javax.swing.*;

JMenuBar menuBar = new JMenuBar();
JMenu fileMenu = new JMenu("File");
JMenuItem openItem = new JMenuItem("Open");
JMenuItem exitItem = new JMenuItem("Exit");

fileMenu.add(openItem);
fileMenu.add(exitItem);
menuBar.add(fileMenu);

frame.setJMenuBar(menuBar);
```
